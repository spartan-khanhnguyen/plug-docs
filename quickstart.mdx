# Get Started with PlugnPay

This guide will help you integrate PlugnPay into your application quickly and start processing payments.

## Step 1: Create product

<AccordionGroup>
  <Accordion icon="key" title="Create your PlugnPay account">
    1. Visit [PlugnPay Dashboard](https://plugnpay.dev) and create your account
    2. Navigate to the Products page
    3. Enter product information, set up feature and pricing
  </Accordion>
</AccordionGroup>


## Step 2: Sync with Stripe

<AccordionGroup>
  <Accordion icon="key" title="Syncing">
    1. Navigate to the Stripe Connection section in your dashboard
    2. Select the environment you want to use: Sandbox for testing, Production for real sales
    3. Enter your Stripe API key and Webhook key (optional) to sync
  </Accordion>
</AccordionGroup>
<Note>
  Your Stripe API keys will be encrypted and be safe
</Note>

## Step 3: Get PlugnPay API Key

<AccordionGroup>
  <Accordion icon="key" title="Get API key">
    1. Navigate to the API keys section in your dashboard
    3. Copy or re-generate API keys for usage
  </Accordion>
</AccordionGroup>

<Warning>

If you re-generate API keys, you must update your client's sdk implementation to use new key.

</Warning>

## Step 4: Install SDK

Choose your preferred programming language and install the corresponding SDK:

<AccordionGroup>
  <Accordion icon="code" title="JavaScript/Node.js">
    ```bash
    npm install @plugnpay/sdk
    ```
  </Accordion>
  <Accordion icon="code" title="Python">
    ```bash
    pip install plugnpay
    ```
  </Accordion>
  <Accordion icon="code" title="Go">
    ```bash
    go get github.com/plugnpay/plugnpay-go
    ```
  </Accordion>
  <Accordion icon="code" title="Java">
    ```xml
    <dependency>
        <groupId>dev.plugnpay</groupId>
        <artifactId>plugnpay-java</artifactId>
        <version>1.0.0</version>
    </dependency>
    ```
  </Accordion>
</AccordionGroup>
